## 1、C++ 编程简介

基于对象(Object Based)
* class without pointer members -Complex
* class with pointer members -String

面向对象(Object Oriented)
* 学习Classes之间的关系
  * 继承(inheritance)
  * 复合(composition)
  * 委托(delegation) 

### C++历史
C语言(1972)
C++(1983)早期叫C with Class

C++有很多版本,如 C++98,C++03,C++ 11,C++14

学习C++(其他语言也是)可以分为两个部分：
* C++语言
* C++标准库
